# SportGear

SportGear e-commerce skeleton with:

- React frontend (`frontend/`)
- Spring Boot backend (`backend/`)
- Technical docs (`docs/`)

Implemented scope in this phase:

- FR-01: Register
- FR-02: Login
- FR-04: Forgot/Reset password
- FR-05: User profile update
- NFR-03: HTTPS-ready transport configuration
- NFR-04: JWT + BCrypt authentication security baseline
- BR-04: Locked account blocked from login and checkout

Quick run:

1. Frontend: see `frontend/README.md`
2. Backend: see `backend/README.md`
3. Security transport notes: see `docs/setup/security-transport.md`
