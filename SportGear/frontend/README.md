# Frontend

React (Vite) app for auth and profile flows.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Default URL: `http://localhost:5173`

## Test

```bash
npm test
```

## Environment

Create `.env` if needed:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

## Implemented routes

- `/register`
- `/login`
- `/forgot-password`
- `/reset-password`
- `/profile`
